# 三、R语言基础

```R
## 创建一个向量：
vec = c(1, 4, 4, 3, 2, 2, 3) # 下标从1开始

## 返回向量中的元素
vec[c(2, 3, 4)] ## 取下标为2， 3， 4的元素
# out：4 4 3
vec[2:4]   ## 取下标从2到4的元素
# out: 4 4 3
vec[c(2, 4, 3)]
# out 4 3 4

## 删除向量中的元素
vec[-2]  # 删除第二个元素
# out：1 4 3 2 2 3
vec[-2:-4] # 删除下标从2到4的所有元素
# out：1 2 2 3

## 提取满足条件的元素
vec[vec<3] # 返回vec中小于3的所有元素
# out: 1 2 2

## 找到元素
which(vec==3) # 返回等于3的下标
# out: 4 7
which.max(vec) # 返回vec的最大值的下标
# out: 2
which.min(vec) # 返回vec的最小值的下标
# out: 1
```
